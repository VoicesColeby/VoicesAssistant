I can give you the command structure for the message_talent.py script.

The script relies on environment variables for its inputs, so you'll set these in your terminal before running the Python file.

Command Structure
You'll need three main pieces of information: the starting URL, the message you want to send, and the remote debugging URL for Chrome.

Launch Chrome in Debugging Mode:
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-address=127.0.0.1 --remote-debugging-port=9222 --user-data-dir="C:\temp\cdp"

Set Environment Variables:

START_URL: The URL of the job responses page. This is where the script will begin its work.

MESSAGE: The actual text you want to send to each talent.

DEBUG_URL: The address of the remote debugging session you started in the first step.

PowerShell

$env:START_URL="https://www.voices.com/client/jobs/responses/818318"
$env:MESSAGE="Hi there, we'd like to move forward with your audition. Please reply to this message to continue."
$env:DEBUG_URL="http://127.0.0.1:9222"
Run the Script:
python message_talent.py
